SQL Migration Optimizer

Convert SQL code from Oracle and SQL Server to PostgreSQL with optimization recommendations

A supplementary tool to AWS DMS and AWS SCT for SQL code conversion and optimization. Use DMS for data migration and SCT for schema conversion — use this tool for fine-tuning SQL code, analyzing query performance, and getting optimization recommendations for PostgreSQL and Aurora.
